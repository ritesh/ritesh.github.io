Run Google Wave Inside Thunderbird 3

Posted on December 11, 2009 in google, thunderbird, wave

Google’s Wave tool suffers from a “just another inbox” problem. So it’s great to see Wave running inside a tab in Thunderbird, right next to your email inbox, with a single JavaScript line. Among Thunderbird 3’s new features is Content Tabs, a way to monitor other web sites and web-based messaging from inside Thunderbird. The Quetzalcoatal blog hacked together a little code that opens Wave cleanly and conveniently inside one of these tabs, and even keeps it open after you restart Thunderbird.
