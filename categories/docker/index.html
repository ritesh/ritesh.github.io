Building Dockerized Application Security Scanners

Posted on July 10, 2015 in docker, docker-compose

I read an interesting post by @anshuman_bh on automating ZAP scans for dockerized web applications. At the time, I was exploring docker compose (formerly fig) as a way to “container-ize” a simple Python web application (Flask, Postgres, RabbitMQ). It occurred to me that docker-compose would be a good way to accomplish this goal. The advantage of using docker-compose over using docker directly is that you can avoid writing “glue” code by defining links between containers.
