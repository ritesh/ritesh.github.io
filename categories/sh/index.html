Do batch things well

Posted on December 4, 2011 in linux, sh

I sometimes forget this really easy one liner for doing things in a batch on Linux/MacOS. # for file in `find . -type f`; do somescript.py $file; done If you’re relying on the output of find it is probably best to use the xargs option to find like so: # find . -type f -print0 | xargs -0 somescript.py Even better, you can avoid using xargs, instead use the exec option to find.
